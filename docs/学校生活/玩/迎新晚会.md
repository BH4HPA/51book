# 迎新晚会

每年，新高二的社团成员都会在团委学生会的带领下，为新入学的新高一举办迎新晚会。

## 活动信息

**活动时间** 入学一周左右。

**活动地点** 一中八百人会议室。

**参加对象** 新高一、高二社团成员、学生会。

每个社团将在本次活动中出一个节目左右，展示社团风貌，吸引有兴趣的同学加入社团。

## 活动内容

晚会内容大致包括唱歌、跳舞、魔术、器乐演奏和游戏环节。
