# 运动会

::: tip 待补充内容
本块有部分内容等待补充。
:::

为了丰富同学们的课余生活，提高同学们的综合素质，五十一中学每年举办一届田径运动会。全体运动健儿本着“青春、奥运、健康”的宗旨，发挥“更高、更快、更强”的体育精神，赛出风格，赛出水平，充分展示了学校体育教育的成果和体育运动水平，展示了同学们顽强拼搏、超越自我的时代风采。

(_摘自《温州市第五十一中学 2019 新生导航手册》_)

## 活动信息

**活动时间** 十月下旬左右。

**活动地点** 一中大操场。

**参加对象** 除可能出去集训的高三艺术班外的全体学生。
